노드는 자바스크립트를 알아듣는 프로그램
---
## 역할
1. 리액트 코드를 자바스크립트로 변환해준다.
## 설치
1. 홈페이지에서 다운로드
2. $node -v
3. Install git
4. Install code editor
5. Open project folder
6. Open terminal(GitBash)
7. hello.js
```js
console.log("Hello");
```
8. $node hello.js

## REPL
1. To start, enter:
	`$node`
- _ refers to the last evaluated value in REPL.
- .help shows available commands.
- .editor enables longer code by providing execute key.
- .save saves the code as a file.
- .exit to exit.
