## 팁
1. 오른쪽 툴바(N) -> View -> 3D 커서 위치 조절 가능
2. 크기나 모양은 편집 모드에서 바꾸자.
3. 노말 재계산 : Mesh -> Normal -> Recalculate outside
4. 오른쪽 위 원 두개의 메뉴를 눌러 노멀 방향이나 노멀 색깔을 확인 가능
5. 노멀 색깔(Face Orientation)
6. 얇은 면을 뽑아낼 때에는 루프를 추가해서 얇은 입체를 뽑아내자.

## 단축키
Shift - A            오브젝트 추가
Shift - D            오브젝트, 점, 선, 면 복사
Z                     3D Mode
Tab                   Object Mode <-> Edit Mode
F                     이어진 면 생성
Shift - N            노멀 재계산
