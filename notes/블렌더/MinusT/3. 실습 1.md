## 팁
1. E 누르고 오른쪽 클릭하면 새로운 점이 겹쳐진 상태로 생성 됨. X 눌러서 점을 삭제.
2. 와이어 프레임 뷰는 뒤쪽에 있는 점들도 선택 가능해 편리
3. Array 모디파이어는 물체 복사, Constant Offset 프로퍼티는 물체 크기에 비례해서 복사된 물체 배치
4. Curve 모디파이어는 Curve 오브젝트를 선택해 물체를 Curve 모양에 따라 배치 가능.
5. Curve 오브젝트 -> N -> Weight, Radius, Tilt를 조절해 커브에 배치된 물체에 영향을 줄 수 있음
6. 모디파이어를 Apply해서 세부 조절 가능(모디파이어를 완전히 완료했을 때만)
7. 머터리얼은 적용되고 있는 물체 숫자도 표시함. 숫자를 클릭하면 독립적인 머터리얼이 됨.(해당 물체 1개를 적용하는 머터리얼)
