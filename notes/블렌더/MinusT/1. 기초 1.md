## UI
1. 카메라 - 렌더링
2. 프린터 - 렌더링 출력(크기, 비율, 파일 종류 둥)
3. 지구본 - 씬의 배경

## 팁
1. 프로퍼티의 트랜스폼(위치, 회전, 크기)를 드래그해서 동시에 편집 가능
2. 하나의 엣지에 면은 2개 까지(그 이상은 노말이 이상해짐)
3. 전체 선택 -> Ctrl - F -> Shader Smooth
